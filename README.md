# Vortigern Store
Create Interactive Mobile UI with React Native

## How to run 

Install React Native CLI
```sh
npm install -g react-native-cli
```

Install Dependencies
```sh
npm i
```

Run react native project
```sh
npm start
```

Run on Andorid Virtual Device
```sh
npm android
```
